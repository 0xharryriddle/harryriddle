## Introduction

The Polkadot.js API is the primary JavaScript/TypeScript library for interacting with Substrate-based blockchains. In this tutorial, you'll learn the fundamentals of connecting to a node and performing basic queries.

## Prerequisites

- Node.js v18+ installed
- Basic TypeScript knowledge
- A code editor (VS Code recommended)

## Step 1: Project Setup

Create a new project and install the required dependencies:

```bash
mkdir polkadot-js-starter && cd polkadot-js-starter
npm init -y
npm install @polkadot/api typescript ts-node
npx tsc --init
```

## Step 2: Connect to a Node

Create a file `src/index.ts`:

```typescript
import { ApiPromise, WsProvider } from '@polkadot/api';

async function main() {
  // Connect to the Polkadot mainnet
  const provider = new WsProvider('wss://rpc.polkadot.io');
  const api = await ApiPromise.create({ provider });

  // Retrieve chain info
  const chain = await api.rpc.system.chain();
  const lastHeader = await api.rpc.chain.getHeader();

  console.log(`Connected to ${chain}`);
  console.log(`Latest block: #${lastHeader.number}`);

  await api.disconnect();
}

main().catch(console.error);
```

// Get account balance
